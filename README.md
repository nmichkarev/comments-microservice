## Project setup

```bash
$ npm install
$ docker compose up
```

## Compile

```bash
$ npm run build
```

## Run

### Run comments microservice
```bash
$ npm run start-service
```

### Run api-gateway
```bash
$ npm run start-service
```

Api docs are available on [http://localhost:3000](http://localhost:3000)

Application uses 3000, 3001 and 5433 ports so it should be free on your computer.
